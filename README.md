# KING METAL v3

## 🚀 تشغيل المشروع محليًا
```bash
npm install
npm run dev
```

## 🛠️ بناء المشروع
```bash
npm run build
npm run preview
```

## 🌐 رفع على GitHub
1. أنشئ Repo جديد على GitHub.
2. اربطه باستخدام git remote.
3. ادفع الكود: `git push -u origin main`.

## ☁️ نشر على Vercel
- اربط الريبو بـ Vercel.
- هيتعرف تلقائيًا على Vite.
- Build command: `npm run build`
- Output: `dist`
